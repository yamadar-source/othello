# Othello (オセロ)

シンプルなブラウザ版オセロです。

使い方:

- `index.html` をブラウザで開くだけで動きます。
- 「コンピュータと対戦」にチェックを入れると白がAIになります（簡易AI）。
- リスタートボタンで新しいゲームを開始します。

ローカルで簡易サーバを立てる場合（推奨）:

```zsh
# カレントを othello ディレクトリにして
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

ファイル:

- `index.html` - UI
- `style.css` - スタイル
- `main.js` - ゲームロジックとUI操作

ライセンス: パーソナル/学習利用向け。改変・再配布可。
